## Instalação

Para esse projeto, foi utilizado o ambiente node.js e necessita, portanto, ter o recurso para execução.

### Pré-requisitos
- [Git](https://git-scm.com/downloads) - v2.44
- [Node.js](https://nodejs.org/en/download/package-manager) - v20.11.1
> Siga os passos de instalação fornecidos pelos sites de download

### Passos para a instalação

1.  Clone o repositório
``` bash
git clone https://github.com/JoaoClaudioSilva/desafio-target.git
cd desafio-target
```

2.  Instale as dependências
``` bash
npm install
```

## Uso
Execute os códigos-fontes desejados utilizando o cmd, da seguinte forma:
``` bash
node tecnica1 <numero>
node tecnica1 <string>
```
> Os valores para execução podem ser alterados nos campos destacados, caso nenhum valor seja especificado serão atribuidos valores padrão.
